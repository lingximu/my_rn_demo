my-rn-demo
---
> 一个练习的作品，前端用 react-native / mbox 、后端用 koajs、数据库用 mysql、接口用 graphql。

Usage
---

```shell
npm install -g yarn react-native-cli@2.0.1
yarn
# android 需要先打开 Android虚拟机或用电脑连接 Android 手机
react-native run-android
# ios 需要在 mac 电脑
react-native run-ios
```
